# Quantum Sentinel 🌌

Advanced holographic APK scanner with quantum visual effects for cybersecurity analysis.

## Features
- 🌌 Quantum holographic interface
- 🔮 3D rotating scanner rings
- 📊 Real-time threat analysis
- 🎨 Liquid glass effects
- 📱 Responsive design
- 🚀 Deployed on Render
